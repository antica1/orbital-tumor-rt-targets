# 眼眶侵犯程度分期 — 眼科 vs 五官科/颌面外科双视角对比

> 文献来源：AJCC 8th Ed. Uveal Melanoma; Oral Oncol 2021 (PMID:33517131); J Craniomaxillofac Surg 2017 (PMID:28011179); Eur Arch Otorhinolaryngol 2024 (PMID:38914815)

## 核心区别

| | 眼科视角 | 五官科/颌面外科视角 |
|---|---|---|
| **肿瘤来源** | 眼内（葡萄膜、视网膜） | 眶外（鼻腔、鼻窦、颌骨、颅底） |
| **侵犯方向** | **从内向外**：眼球 → 巩膜外 → 眼眶 | **从外向内**：鼻窦/鼻腔 → 眶壁 → 眶骨膜 → 眶脂肪 → 眼外肌 |
| **保眼定义** | 保留眼球本身（不摘除） | 保留眶内容物（不剜除） |
| **代表分期** | AJCC 葡萄膜黑色素瘤 T 分期 | 鼻窦恶性肿瘤眼眶侵犯 I-IV 级 |
| **核心问题** | 肿瘤是否已穿破巩膜？穿出多少？ | 眼眶侵犯深度到哪一层？能否保眼？ |

## 一、眼科视角：AJCC 葡萄膜黑色素瘤 T 分期（从内向外）

### AJCC 8th Edition (2017) — 睫状体和脉络膜黑色素瘤

| T 分期 | 肿瘤基底径 (mm) | 厚度 (mm) | 关键特征 |
|--------|---------------|----------|---------|
| **T1** | ≤12.0 | ≤6.0 | 眼球内局限，无巩膜外侵犯 |
| **T2** | 12.1-18.0 | 6.1-10.0 | — |
| **T3** | >18.0 | 10.1-12.0 | 大型肿瘤，仍局限 |
| **T4a** | 任意 | 任意 | **巩膜外侵犯 ≤5 mm** |
| **T4b** | 任意 | 任意 | **巩膜外侵犯 >5 mm** |

### 虹膜黑色素瘤

| T 分期 | 特征 |
|--------|------|
| T1 | 局限于虹膜 |
| T2 | 侵犯睫状体和/或脉络膜 |
| T3 | 侵犯巩膜 |
| T4a | 巩膜外侵犯 ≤5 mm |
| T4b | 巩膜外侵犯 >5 mm |

## 二、五官科/颌面外科视角：鼻窦恶性肿瘤眼眶侵犯 I-IV 级（从外向内）

| 分级 | 侵犯深度 | MRI 表现 | 能否保眼 |
|------|---------|---------|---------|
| **Grade I** | 肿瘤**贴近**眶骨膜，未侵蚀 | 眶壁骨质完整或仅受压变薄；眶骨膜完整，可见脂肪线 | ✅ 可保眼 |
| **Grade II** | 肿瘤**侵蚀眶壁骨质**，眶骨膜完整但受侵 | 骨质缺损；眶骨膜可增厚/强化，但仍可见脂肪间隙 | ✅ 可保眼，需连同眶骨膜一同切除 |
| **Grade III** | 肿瘤**穿透眶骨膜**，侵犯眶内脂肪 | 眶内脂肪被肿瘤推移/浸润，眼外肌尚未受累 | ⚠️ 争议区域 |
| **Grade IV** | 肿瘤侵犯**眼外肌、眶尖、视神经** | 眼外肌轮廓不清，眶尖脂肪消失，视神经被包绕 | ❌ 基本需剜除 |

### 保眼的关键决策线

- Grade I: 手术完整切除 + PORT → ✅ 保眼
- Grade II: 骨膜一并切除 + PORT → ✅ 保眼
- Grade III: 脂肪受累局限 → ⚠️ 可保眼+放疗推量；广泛 → ❌ 剜除
- Grade IV: 眼外肌/眶尖/视神经 → ❌ 剜除

## 三、两种分期的底层逻辑对比

| 维度 | 眼科 AJCC | 五官科/颌面外科 I-IV |
|------|----------|--------------------|
| **侵犯方向** | 眼球 → 眼眶（单向） | 鼻窦 → 眼眶（单向） |
| **关键屏障** | 巩膜 | 眶骨膜（Periorbita） |
| **核心影像模态** | 间接检眼镜 + 超声 + MRI | MRI T1+C 脂肪抑制 |
| **放疗角色** | 质子/SRS（眼内小肿瘤）；手术摘除后+辅助 | IMRT 术后辅助（Grade II/III 保眼后） |
| **淋巴转移风险** | 仅巩膜外侵犯后（T4）→ 眶外→有 | Grade III/IV 突破眶壁后→有 |

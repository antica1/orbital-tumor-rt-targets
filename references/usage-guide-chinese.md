# Orbital Tumor RT Skill — Usage Guide (Chinese)

## 核心决策链
1. 三间室定位（眼内/眶内/眶外→淋巴评估）
2. 肿瘤类型→剂量
3. 手术状态→靶区范围（R0标准，R1/R2间室放疗）
4. 分级→间室延伸（低级照门50-54Gy，高级门+隔壁60-66Gy）
5. 淋巴结→VIII/IX/IIa瀑布
6. 眼前庭保护→角膜/泪腺/鼻泪管+眼科共管

## 术语规范
- 泪腺（lacrimal gland）——解剖腺体，不是"泪膜"
- 眼前庭——眼睑+结膜+角膜+泪腺+鼻泪管这组前部结构
- 间室放疗——照全受累间室，不是GTV+Nmm

## 三大原创贡献
- 间室放疗理念（术后/颅底→照全间室+下一间的门）
- 眶外淋巴瀑布（VIII→IX→IIa）
- 眼前庭全程共管（角膜/泪腺/鼻泪管+眼科时间表）

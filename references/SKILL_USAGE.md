# 眼眶肿瘤放疗靶区勾画 Skill — 使用说明

> `orbital-tumor-rt-targets` v1.1 | 朱国培 · 上海九院

## 核心概念速查

### 间室放疗 vs 距离放疗

| | 距离原则（鼻咽癌） | 间室原则（眼眶/颅底） |
|---|---|---|
| 什么时候用 | 根治性 RT，未手术 | 术后、R1/R2、颅底 |
| CTV 逻辑 | GTV + N mm | 整个受累间室 |
| 漏照代价 | 边缘复发→可手术 | **边缘复发→不可挽救** |

### "门和隔壁房间"

```
眼眶（已知受累）→ 门：眶上裂/视神经管 → 隔壁：海绵窦
  低级别：照门（50-54 Gy 预防）→ 砌防火墙
  高级别：门已被破 → 隔壁也照（60-66 Gy 根治）
```

### 无锯骨 = 无 R0

眼科医生不切骨。手术记录没写"截骨/开颅/上颌骨切除"→ 骨缘按 R1/R2 处理 → PTV-66 + 包全骨段。

### 眶外淋巴瀑布

眼睑/结膜/眶周皮肤受累 → VIII（腮腺浅叶，眼科最易漏）→ IX（面动脉旁）→ IIa

### 术语规范

- **眼前庭**（anterior orbit）：眼睑+结膜+角膜+泪腺+鼻泪管——非"眼前节"
- **泪腺**（lacrimal gland）：分泌泪液的腺体，放疗后干眼症的原因——非"泪膜"

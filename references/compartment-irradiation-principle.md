# 间室放疗与"门"概念 — 跨技能核心原则

> 起源：眼眶肿瘤 Skill (Section 2.6.5-2.6.6) | 推广至：NPC 靶区、口腔/口咽术后靶区、HNCUP 靶区

---

## 核心定义

### 间室放疗 (Compartment Irradiation)

当肿瘤侵犯或手术扰动了一个解剖间室（筋膜间隙、肌肉间隙、神经血管通道），放疗的 CTV 必须覆盖**整个间室**，而不是仅扩 GTV 周围的几毫米。

| 情景 | CTV 范围 |
|------|---------|
| 肿瘤在间室内 + 手术仅切了间室的一部分 | **全间室** |
| 肿瘤未在该间室 + 但手术分离/翻开了该间室 | **全间室**（播散风险） |
| 完整的间室切除 (en-bloc) + 切缘阴性 | 不照 |

### "门" (Door / Barrier) 概念

两个相邻间室之间的解剖通道或分界——"门"。放疗只需覆盖门（预防剂量），阻断从受累间室向未受累间室的镜下扩散通道。不需要照整个隔壁间室。

```
间室 A（安全）←→ 门（照预防剂量）←→ 间室 B（已受累，照根治量）
```

## 各技能中的门

| 技能 | 门 | 低级别/预防 | 高级别/已受累 |
|------|----|----------|------------|
| **眼眶肿瘤** | 眶上裂 / 视神经管 | 50-54 Gy（眶尖砌墙） | 60-66 Gy + 海绵窦 |
| **眼眶肿瘤** | VIII/IX 淋巴结 | 预防照（颈清后逆流） | 累及时 60-66 Gy |
| **NPC** | 破裂孔 / 卵圆孔 / 岩尖 | 50-54 Gy（偏心时豁免侧可选） | 60-70 Gy |
| **口腔术后** | II/V 交界口（胸锁乳突肌后缘） | 54 Gy（II 阳性时照门槛） | 不照全 V |
| **口腔术后** | 咽侧壁/RP 交界 | 54 Gy（硬腭后 1/2、扁桃体） | 仅外侧组 |
| **HNCUP** | 同侧颈-对侧颈 | 单侧颈（术后逆流时扩至对侧 Ib） | 双侧淋巴结受累时全双侧 |

## 距离原则 vs 间室原则

| | 距离原则（鼻咽癌根治性 RT） | 间室原则（颅底/眶/术后） |
|---|---|---|
| CTV 逻辑 | GTV + N mm | 全受累间室 + 门 |
| 预防依据 | 离 GTV 越远越安全 | 解剖连续性——进了这间就照全 |
| 漏照后果 | 边缘复发→可手术/再程 RT | 边缘复发→不可挽救 |
| 决策口诀 | "离多远算安全？" | "这个结构属于哪个间室？它和下一间的门在哪？" |

---

## 间室放疗的层级

```
Level 1：肿瘤所在的间室 → 照全（高危 CTV，66-70 Gy）
Level 2：手术扰动过的相邻间室 → 照全（中危 CTV，60 Gy）
Level 3：与 L1/L2 通过明确"门"（通道）相连的间室 → 照门（低危 CTV，50-54 Gy）
Level 4：无解剖通道相连的间室 → 不照
```
